HOLIDEC SAC - Sitio one-page (entregable)
Archivos incluidos:
 - index.html
 - styles.css
 - script.js
 - assets/logo.svg
 - assets/placeholder1.svg

Instrucciones rápidas:
1) Prueba local:
   - Abre index.html en tu navegador para ver la página.
   - Reemplaza los placeholders en /assets por tus fotos (mismo nombre o actualiza src en index.html).

2) Editar textos:
   - Abre index.html con un editor (Visual Studio Code, Notepad++).
   - Cambia el nombre de la empresa, servicios, teléfono, y otros textos directamente.

3) Formulario:
   - Actualmente el formulario usa 'mailto' para abrir tu cliente de correo.
   - Para formulario profesional, crea una cuenta en Formspree (https://formspree.io) o usar un backend.
   - Reemplaza la función submitForm en script.js con la llamada fetch a tu endpoint.

4) Subir la web (opción rápida):
   - Crea una cuenta en Netlify (https://app.netlify.com) o Vercel y arrastra la carpeta descomprimida.
   - O usa GitHub Pages: sube los archivos a un repo y activa Pages en 'gh-pages' branch o main.

5) Dominio:
   - Compra un dominio en Namecheap/GoDaddy/Hostinger y conéctalo desde Netlify o tu hosting.

6) Sugerencias para mejorar (puedo ayudarte):
   - Integrar un formulario funcional (Formspree / Google Forms).
   - Reemplazar las imágenes por fotos reales y optimizarlas.
   - Animaciones adicionales y SEO básico (meta tags, structured data).

Si quieres, puedo:
 - Preparar 4 imágenes optimizadas para reemplazar los placeholders.
 - Configurar el archivo para que use Formspree y esté listo para producción.
 - Crear la versión lista para WordPress si prefieres migrar después.
